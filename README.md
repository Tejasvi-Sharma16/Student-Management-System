# Student-Management-System
## Project Description
This is a beginner - friendly C++ project that demonstrates the use of Object-Oriented Programming (OOP) concepts such as classes, objects, and encapsulation .

The project allows users to:
- Add student details (Roll number, Name, Marks)
- Display student details
- Manage student records in a simple and organized way

This system can help in managing student information efficiently, innstead of handling it manually.

## Features
- Input and store student data
- Display student information
- Menu-based system for easy navigation

## Tech Stack
- C++
- Object-Oriented Programming (OOP)
